## Ejercicio 1

### Enunciado

Ejercicio 1. Sea Σ = {𝑎, 𝑏} un alfabeto. Hallar:
Σ0, Σ1, Σ2, Σ∗, Σ+, |Σ|, |Σ0|
(|𝐴| indica la cantidad de elementos de 𝐴).

### Resolucion

```
Σ0 -> {λ}
Σ1 -> Σ
Σ2 -> {aa,ab,ba,bb}
Σ∗ -> {λ, a,b,aa,ab,ba,bb,aaa,aab,aba,bab, (...)}
Σ+ -> Σ* \ {λ}
|Σ| -> 2 (λ NO pertenece al alfabeto)
|Σ0| -> 1
```
