## Ejercicio 7

### Enunciado

Definir por comprensión los siguientes lenguajes:

a. ℒ1 = {𝑎𝑏, 𝑎𝑎𝑏𝑏, 𝑎𝑎𝑎𝑏𝑏𝑏, …}

b. ℒ2 = {𝑎𝑎𝑏, 𝑎𝑎𝑎𝑎𝑏𝑏, 𝑎𝑎𝑎𝑎𝑎𝑎𝑏𝑏𝑏, …}

c. ℒ3 = {𝑎𝑎𝑎𝑏𝑐𝑐𝑐, 𝑎𝑎𝑎𝑎𝑏𝑐𝑐𝑐𝑐, 𝑎𝑎𝑎𝑎𝑎𝑏𝑐𝑐𝑐𝑐𝑐, …}

(donde el «crecimiento» en la cantidad de cada símbolo es lineal en cada caso)

### Resolucion

a. { a^n.b^n | n ≥ 1 }
b. { a^n.b^m | n = 2m ∧ m ≥ 1 }
c. { a^n.b.c^n | n ≥ 3 }
